Bienvenido al Bot de Discord

Para añadir el bot a tu servidor de Discord, haz clic en el siguiente enlace:

Añadir Bot

Instrucciones para hacer funcionar el bot:

  • 1. Asegúrate de tener Node.js y npm instalados.
  • 2. Clona el repositorio del bot o copia los archivos a tu máquina.
  • 3. Navega a la carpeta del bot y ejecuta npm install.
  • 4. Ejecuta el bot con node bot.js.
  • 5. Usa el comando !amor, !sexo, etc. para interactuar con el bot.
